### degree
```
Computes the degrees for each node 
Input:
- n: 
- mode:
Output:
file
Example
Compute the interactors of the HomoSapiens_intact_network associated with the 20 largest degrees in local mode:

java -jar diamin-1.0.0-all.jar LOCAL human_intact_network.txt degree 20

Compute the interactors of the HomoSapiens_intact_network associated with the 20 largest degrees in cluster mode:
```
